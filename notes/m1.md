## 核心心智模型

pandas 操作 = 用程式碼完成 Excel 的操作（新增欄位、篩選、取值...）

最重要的是理解資料經過操作後的結構：
- 通常不是保留 DataFrame 就是變成 Series
- 兩個物件有各自對應的 methods
- chain operation 的閱讀理解 = 考驗資料在過程中變形的掌握程度

## API 整理

### 1. Mask 篩選
把一連串條件包進去，通常能拿到想要的結果。
```python
df[(condition1) & (condition2)]   # AND
df[df['col'].isin([v1, v2])]      # 在清單內
df[布林Series]                    # 篩出 True 的列
```

### 2. set_index
把想要的欄位抓出來作為 index（如把時間欄變成 index）。
設為 DatetimeIndex 後，開啟一整套時序操作能力：

```python
df.set_index('datetime')

# Partial String Indexing（只有 DatetimeIndex 才支援）
df.loc['2015-03']                      # 取出整個 3 月
                                       # 內部展開成 df.loc['2015-03-01 00:00:00':'2015-03-31 23:59:59']
df.loc['2015-03-01':'2015-03-07']      # 取出一週
df.loc['2015-03-01':'2015-03-07', 'volt']  # 取出一週的 volt 欄

# resample 和 rolling 也都需要 DatetimeIndex
df.resample('D').mean()
df.rolling('24h').mean()               # M4 rolling feature 的核心
```

### 3. 取值
```python
df['col']               # → Series
series['index']         # → 單一值
df.loc[3, 'volt']       # index 標籤是 3 的那列，volt 欄
df.iloc[3, 2]           # 第 4 列（位置3），第 3 欄（位置2）
```

### 4. 分組聚合（groupby）
樞紐分析表概念，對某組進行某欄的統計分析。
```python
df.groupby('machineID')['volt'].agg(['mean', 'std'])
df.groupby('machineID').size()
```

### 5. Resample — 時序重採樣
pandas 專為時序設計的 groupby，以時間頻率為分組依據。
```python
df.set_index('datetime').resample('D').mean()   # 每日平均
df.set_index('datetime').resample('W').size()   # 每週筆數
df.set_index('datetime').resample('ME').sum()   # 每月加總
```

### 6. Merge — 表格合併
把兩張表按 key 欄對齊合併。
```python
pd.merge(left, right, on='machineID', how='left')   # left join，保留左表所有列
pd.merge(left, right, on='machineID', how='inner')  # inner join，只保留兩邊都有的 key
```
